# 🎭 Personagens para Roleplay (RP)

Bem-vindo ao meu arquivo central de personagens. Este repositório serve como uma biblioteca detalhada para consultar fichas, históricos e atributos técnicos para diferentes cenários de RP.

---

## 📂 Organização do Repositório

Os personagens estão divididos por categorias ou gêneros de cenário. Navegue pelas seções abaixo para encontrar a ficha ideal:

* **Fantasia Medieval:** Guerreiros, magos e criaturas místicas.
* **Cyberpunk / Sci-Fi:** Hackers, mercenários e sintéticos.
* **Moderno / Realista:** Cotidiano, drama e investigação.
* **Sobrenatural:** Vampiros, lobisomens e caçadores.

---

## 📝 Estrutura das Fichas

Cada personagem neste repositório segue este padrão de organização:

1.  **Conceito:** Uma frase que define a essência do personagem.
2.  **Personalidade:** Traços comportamentais, medos e motivações.
3.  **Habilidades:** O que o personagem sabe fazer (ou poderes, se houver).
4.  **Histórico:** Um resumo do passado que moldou quem ele é hoje.

## 🛠️ Como Usar

Se você é meu parceiro de RP, sinta-se à vontade para:
* Ler as fichas para entender a dinâmica de grupo.
* Sugerir conexões de backstory entre nossos personagens.
* Verificar os limites e gatilhos (Triggers) listados em cada ficha específica.

> [!IMPORTANT]
> **Nota de Uso:** Estes personagens são de autoria própria. Por favor, não os utilize em outros RPs sem autorização prévia.
